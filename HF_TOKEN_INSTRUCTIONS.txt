        HOW TO CREATE HUGGING FACE ACCESS TOKEN
═══════════════════════════════════════════════════════════════

STEP 1: Go to Hugging Face Token Settings
→ https://huggingface.co/settings/tokens

STEP 2: Create New Token
1. Click "New token" button
2. Name: "Backend Deployment" (or any name you prefer)
3. Type: Select "Write" (needed to upload files)
4. Click "Generate token"

STEP 3: Copy the Token
⚠️ IMPORTANT: Copy the token immediately!
   It will only be shown once.

   The token looks like: hf_xxxxxxxxxxxxxxxxxxxxxxxxxxxxx

STEP 4: Provide Token to Claude
Once you have the token, paste it in the chat and I'll:
- Authenticate with Hugging Face
- Upload all 29 backend files
- Configure the Space
